# neptune-r

R interface for [Neptune](http://neptune.ai/), machine learning experiment tracking tool.

It lets you easily track:
* hyperparameters
* metrics
* performance charts and images
* model binaries

Everything is logged to Neptune and can be:
* organized
* visualized
* shared 
* accessed
